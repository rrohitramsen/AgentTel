AgentTel is a JVM library that enriches OpenTelemetry spans with the structured context AI agents need to autonomously diagnose, reason about, and resolve production incidents.

Modules

  • agenttel-api — Annotations, attribute constants, enums, and data models
  • agenttel-core — Runtime engine: span enrichment, baselines, anomaly detection, SLO tracking
  • agenttel-genai — GenAI instrumentation: LangChain4j, Spring AI, Anthropic/OpenAI/Bedrock SDKs
  • agenttel-agent — Agent interface layer: MCP server, health, incidents, remediation
  • agenttel-javaagent-extension — Zero-code OTel javaagent extension
  • agenttel-spring-boot-starter — Spring Boot auto-configuration
  • agenttel-testing — Test utilities for verifying span enrichment

See the project README for usage documentation.
